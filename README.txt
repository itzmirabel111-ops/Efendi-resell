EFENDI – GitHub Pages Quickstart
================================

So geht's (2 Minuten):
1) Neues Repo erstellen (Public), z. B. efendi-shop
2) ALLE Dateien aus diesem ZIP **entpacken** und in das Repo hochladen (nicht das ZIP selbst hochladen)
3) Settings → Pages → Source: Deploy from branch, Branch: main, Folder: /(root)
4) Speichern. Link erscheint: https://DEINNAME.github.io/REPO/
5) Warte kurz, dann Seite aufrufen. Wenn du diese Startseite siehst, läuft GitHub Pages korrekt.

Tipps:
- Die Datei `.nojekyll` verhindert, dass GitHub irgendwelche Build-Schritte ausführt.
- Lege deine Bilder in `images/` ab und referenziere sie relativ, z. B. `<img src="images/meinbild.jpg">`.
- Für Unterseiten kopiere `index.html` als Vorlage und passe Inhalte an.
